\### Purpose

To experiment with real-time communication protocol in .NET with SignalR through Console apps. The goal is to test the simplest case of "ping" and "pong" between two consoles, to learn patterns and resources needed to use and set up SignalR.



⚙️ \*\*Tech stack:\*\* C# and .NET 10



⚛️ \*\*App:\*\* Console
